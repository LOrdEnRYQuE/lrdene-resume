# Admin Suite Handover Documentation

Welcome to your Premium Portfolio OS Admin Suite. This guide explains how to manage your content and leads.

## 1. Accessing the Dashboard
Navigate to `/admin` to access the main command center. From here, you can monitor leads and jump to specific management modules.

## 2. Lead Pipeline (`/admin/leads`)
- **Monitoring**: All contact form submissions appear here in real-time.
- **Status Management**: Track leads from `New` to `Contacted` or `Archived`.
- **Details**: Click on a lead to view the full message and contact information.

## 3. Journal Manager (`/admin/blog`)
- **Editorial Control**: Create, edit, and delete blog posts.
- **Categorization**: Tag posts (e.g., Engineering, AI, Design) for automatic filtering.
- **Publishing**: Manage draft states and publication dates.

## 4. Media Library (`/admin/media`)
- **Asset Management**: Centralized hub for uploading images and project assets.
- **Organization**: Grid and List views for easy navigation.
- **Usage**: Use these assets in your blog posts and project dynamic routes.

## 5. Technical Maintenance
- **Backend**: Powered by **Convex**. Any schema changes should be handled in the `/convex` directory.
- **SEO**: Sitemaps and robots.txt are automatically generated based on your content.
- **Performance**: Built with Next.js 16 (Turbopack) for maximum speed.
